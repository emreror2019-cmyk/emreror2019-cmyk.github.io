# DevBots Flask Market v7

Сайт в стиле исходного дизайна с единым живым фоном, маркетом, пользователями, демо-балансом, покупками, арендой ботов, разделом «Вакансии», SQLite-базой и расширенной админ-панелью.

## Запуск в VS Code / Windows PowerShell

```powershell
py -m venv .venv
.\.venv\Scripts\activate
python -m pip install -r requirements.txt
python app.py
```

Сайт: http://127.0.0.1:5000

Вход пользователей и администратора: http://127.0.0.1:5000/login

Админ по умолчанию:

```text
admin
admin123!
```

После первого входа смените пароль администратора в разделе `Настройки`.

## Что исправлено в v7

- убраны видимые переходы и затемнения между блоками сайта;
- добавлен единый живой фон с плавными анимациями на всех секциях;
- из верхнего меню убрана кнопка `Услуги`;
- верхнее меню больше не затемняется и не получает обводку при прокрутке;
- страница `Вакансии` получила такую же верхнюю плашку, как главная;
- выровнена кнопка `Баланс`;
- кнопка `Оставить заявку` сделана компактнее;
- в админке полностью переработан блок статусов покупок: крупные этапы, подсказки и сохранение заметки одним действием.

## Возможности, сохранённые из прошлых версий

- маркет товаров и шаблонов сайтов;
- покупка товара с выбором тарифа `Басик / VIP / Pro`;
- аренда бота с выбором дней;
- цена покупки и аренды меняется от тарифа и срока;
- демо-баланс пользователя;
- личный кабинет;
- загрузка фото карточки и файла выдачи товара;
- демо-ссылки для товаров;
- редактирование товаров, тарифов, калькулятора, FAQ, отзывов, пользователей и заявок;
- SQLite-база данных.

## База данных

SQLite создаётся автоматически:

```text
data/site.db
```

SQLite встроен в Python, отдельно устанавливать базу не нужно.

Файлы загрузок:

```text
uploads/product_images/
uploads/product_files/
```

На хостинге папки `data/` и `uploads/` должны быть доступны для записи.

## Изменения v10 — ChinaBot-inspired redesign

- Главная страница в `dist/index.html` заменена на новый тёмный SaaS/market landing в стиле референса: glassmorphism, neon gradients, live demo card, pipeline, feature cards, pricing и FAQ.
- Корневой маршрут Flask уже отдаёт `dist/index.html`, поэтому новый дизайн применяется сразу после запуска `python app.py`.
- Новый фронтенд сохраняет интеграции с существующими API: `/api/me`, `/api/products`, `/api/order`, `/api/purchase`.
- Для совместимости такой же файл продублирован в `frontend_exact/dist/index.html`.
